When you want to undo the changes that already done, use checkout command

git checkout filename

or

or

After checkout changes will be removed.and it is safe becouse, checkout will not delete or change previous commits

difference between git reset --mixed and git reset --soft

difference>

git reset --mixed

it will delete the previous commits and unstage the files from staging area

git reset --soft

it will delete the previous commits and it maintains the files and doesnt unstage the files
