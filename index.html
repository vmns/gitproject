CheckOut: Undoing changes

When you want to undo the changes that already done, use checkout command

git checkout filename

or

git checkout .

or

git checkout *

After checkout changes will be removed.and it is safe becouse, checkout will not delete or change previous commits

Using checkout we can also delete the previous commits using commit id as below

Git reset

reset command will permanently delete your work
